Default Magicpro Host

To make a custom project

  • Define new host configuration files' path in .env, for example: 
    CONF_HOSTS=./data/config/hosts/
  • Add new host configuration file 
    $ make new.host HOST="magicpro.local"
  • Define path for project files of a web hosts in .env, for example: 
    DATA_HOSTS=./data/web/
  • Add new host project directory, with a minimum required files (public/index.html) 
    [DATA_HOSTS]/magicpro.local/public/index.html
