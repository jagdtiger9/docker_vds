Default Magicpro Host

To make a custom project

  • Define the configuration files' directory in .env, for example: 
    [CONF_HOSTS]=./data/config/hosts/
  • Add a new host configuration file: 
    $ make new.host HOST="magicpro.local"
  • Define the path to the web host project files in .env, for example: 
    [DATA_HOSTS]=./data/web/
  • Add a new host project directory with a minimal set of required files (public/index.html) 
    [DATA_HOSTS]/magicpro.local/public/index.html

Directory structure example

[DATA_HOSTS]                 # DATA_HOSTS - all projects root
├── site1.local/
│   └── public/              # Document root
│       └── index.php
├── site2.local/
│   └── public/
│       └── index.php
└── magicpro.local/
    └── public/
        └── index.php

[CONF_HOSTS]                 # CONF_HOSTS - nginx configs
├── site1.local.conf
├── site2.local.conf
└── magicpro.local.conf
